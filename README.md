# neural-network-diabetes
This project applies a feedforward neural network to predict the onset of diabetes in patients based on 8 clinical measurements from the Pima Indians Diabetes Database (Kaggle). Two architectures are trained and compared: an original lightweight model (32→16→1) and an improved V2 model (64→Dropout(0.3)→32→16→1) with regularization. The V2 model achieves 75.32% accuracy and higher precision, reducing false positives. Built with TensorFlow/Keras, scikit-learn, and Python. Part of a deep learning course assignment on binary medical classification.
